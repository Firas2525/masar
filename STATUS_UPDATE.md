# تحديث عرض حالة الموافقة - Status Update

## 🎯 التحديث المطلوب

تم تعديل عرض حالة الموافقة على الأطفال بحيث:

### ✅ ما يتم عرضه:
- **"في انتظار المراجعة"** - للأطفال الذين حالتهم `wait`
- **"تم الرفض"** - للأطفال الذين حالتهم `rejected`

### ❌ ما لا يتم عرضه:
- **"تمت الموافقة"** - للأطفال الذين حالتهم `approved` (لا يتم عرض أي مؤشر)

## 🔧 التغيير التقني

تم تعديل دالة `_buildEnhancedApprovalStatusIndicator` في ملف `lib/view/home/home_view.dart`:

```dart
case 'approved':
  // لا يتم عرض أي مؤشر لحالة الموافقة
  return SizedBox.shrink();
```

## 📊 النتيجة

الآن في الواجهة الرئيسية:
- الأطفال الذين تمت الموافقة عليهم سيظهرون بدون أي مؤشر حالة
- الأطفال في انتظار المراجعة سيظهرون مع مؤشر برتقالي
- الأطفال المرفوضين سيظهرون مع مؤشر أحمر

## 🎨 المظهر النهائي

| حالة الطفل | المؤشر المعروض | اللون |
|-----------|----------------|-------|
| تمت الموافقة | لا يوجد مؤشر | - |
| في انتظار المراجعة | "في انتظار المراجعة" | برتقالي |
| تم الرفض | "تم الرفض" | أحمر |
