# تطبيق الروضة - Kindergarten App

تطبيق Flutter احترافي لإدارة الروضة مع واجهة مستخدم حديثة وجميلة.

## ✨ المميزات الجديدة

### 🎨 التصميم المحسن
- **ألوان جديدة ومتدرجة**: تم تحديث نظام الألوان ليشمل ألوان أكثر احترافية وجمالاً
- **خطوط محسنة**: استخدام خط GT Sectra Fine للنصوص العربية
- **تأثيرات بصرية**: ظلال متدرجة وتأثيرات حركية جميلة
- **تصميم متجاوب**: يتكيف مع جميع أحجام الشاشات

### 🏠 الواجهة الرئيسية المحسنة
- **خلفية متدرجة**: خلفية جميلة مع دوائر زخرفية متدرجة
- **بطاقات محسنة**: تصميم جديد للبطاقات مع ظلال وحدود دائرية
- **أيقونات حديثة**: استخدام أيقونات Material Design الجديدة
- **حالات التحميل**: مؤشرات تحميل جميلة ومتطورة

### 🎯 الألوان الجديدة
- **الألوان الأساسية**:
  - `colorPrimary`: أزرق بنفسجي جميل (#667EEA)
  - `colorSecondary`: بنفسجي غامق (#764BA2)
  - `colorAccent`: وردي فاتح (#F093FB)
  - `colorSuccess`: أخضر (#4CAF50)
  - `colorWarning`: برتقالي (#FF9800)
  - `colorError`: أحمر فاتح (#E57373)

- **التدرجات اللونية**:
  - `colorGradientPrimary`: تدرج أزرق بنفسجي
  - `colorGradientSecondary`: تدرج وردي
  - `colorGradientSuccess`: تدرج أخضر
  - `colorGradientWarning`: تدرج برتقالي

## 🎨 نظام الألوان

تم تحديث نظام الألوان ليكون أكثر احترافية:

```dart
// الألوان الأساسية
const colorPrimary = Color(0xFF667EEA);
const colorSecondary = Color(0xFF764BA2);
const colorAccent = Color(0xFFF093FB);

// التدرجات
const colorGradientPrimary = LinearGradient(
  colors: [Color(0xFF667EEA), Color(0xFF764BA2)],
  begin: Alignment.topLeft,
  end: Alignment.bottomRight,
);
```
